The Prison Coder

Code written from the inside.

This site recreates a project I first built at San Quentin State Prison in 2019, with the help of free staff who believed coding should be accessible everywhere.

It exists to give prison coders open-source code they can study, print, and use to understand how real enterprise applications are built.

Code 1 is Bookworm — an open-source eBook reader written in Vala and GTK. The PDF contains 8 core source files (64 pages) showing how a real desktop app is structured: entry point, application class, data model, database layer, UI, and format readers. Source: github.com/babluboy/bookworm
